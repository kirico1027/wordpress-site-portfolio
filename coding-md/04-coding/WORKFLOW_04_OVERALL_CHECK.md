# 工程 4 補足: 全体チェック

## 概要

HTML、CSS、全体の整合性をチェックし、最終的な品質を確保します。

## 前提条件

- HTML と SCSS ファイルが作成されていること

## チェック項目

### 1. HTML と CSS の整合性

- [ ] HTML で使用されているクラスがすべて SCSS に定義されているか
- [ ] SCSS で定義されているクラスが HTML で使用されているか（使用されていないクラスは削除）
- [ ] クラス名の命名が統一されているか

### 2. 画像の整合性

- [ ] HTML で参照されている画像がすべて存在するか
- [ ] 画像パスが正しいか
- [ ] 画像の命名が統一されているか

### 3. リンクの整合性

- [ ] すべてのリンクが正しく機能するか
- [ ] リンク先が適切か（現時点では `#` でも可）

### 4. レスポンシブ対応

- [ ] すべてのセクションでレスポンシブ対応がされているか
- [ ] モバイル表示で問題がないか
- [ ] タブレット表示で問題がないか

### 5. ブラウザ互換性

- [ ] 主要ブラウザ（Chrome, Firefox, Safari, Edge）で表示が正しいか
- [ ] 古いブラウザでの表示に問題がないか（必要に応じて）

### 6. パフォーマンス

- [ ] ページの読み込み速度が適切か
- [ ] 画像の最適化がされているか
- [ ] 不要な CSS や JavaScript が含まれていないか

### 7. アクセシビリティ

- [ ] キーボード操作でナビゲーション可能か
- [ ] スクリーンリーダーで問題なく読み上げられるか（必要に応じて）
- [ ] コントラスト比が適切か

### 8. コード品質

- [ ] コードが読みやすいか
- [ ] コメントが適切に記述されているか（必要に応じて）
- [ ] コードの重複がないか

## チェック手順

1. **HTML と CSS の整合性チェック**

   ```bash
   # HTMLで使用されているクラスを抽出
   grep -o 'class="[^"]*"' {メインHTMLファイル} | sort -u > html_classes.txt

   # SCSSで定義されているクラスを抽出
   grep -o '^\.[a-z][a-z0-9_-]*' {SCSSディレクトリ}/**/*.scss | sort -u > scss_classes.txt

   # 比較
   diff html_classes.txt scss_classes.txt
   ```

2. **画像の整合性チェック**

   ```bash
   # HTMLで参照されている画像を抽出
   grep -o 'src="\./{画像パス}/[^"]*"' {メインHTMLファイル} | sort -u > html_images.txt

   # 実際に存在する画像を確認
   ls -1 {出力ディレクトリ} | sort > existing_images.txt

   # 比較
   diff html_images.txt existing_images.txt
   ```

**注意**: `{メインHTMLファイル}`, `{SCSSディレクトリ}`, `{画像パス}`, `{出力ディレクトリ}` はプロジェクトごとに異なります。

3. **レスポンシブ対応の確認**

   - ブラウザの開発者ツールでモバイル表示を確認
   - 各セクションで問題がないか確認

4. **ブラウザ互換性の確認**

   - 複数のブラウザで表示を確認
   - 問題があれば修正

5. **問題の修正**

   - 見つかった問題を修正
   - 修正後、再度チェック

6. **最終確認**
   - すべてのチェック項目を確認
   - 問題がなければ完了

## チェックリスト

### HTML

- [ ] DOCTYPE 宣言が正しい
- [ ] セマンティック HTML が使用されている
- [ ] すべての画像に alt 属性が記述されている
- [ ] リンク要素が適切に使用されている
- [ ] BEM 記法に従ったクラス名が使用されている

### CSS

- [ ] BEM 記法に従っている
- [ ] 入れ子が使用されていない
- [ ] rem()関数が使用されている
- [ ] @use が使用されている
- [ ] メディアクエリが適切に記述されている
- [ ] 変数が適切に使用されている

### 全体

- [ ] HTML と CSS の整合性が取れている
- [ ] 画像が正しく表示されている
- [ ] レスポンシブ対応がされている
- [ ] ブラウザ互換性がある
- [ ] パフォーマンスが適切
- [ ] アクセシビリティが考慮されている

## 注意事項

- 見つかった問題は必ず修正すること
- 最終確認で問題がなければ完了とする
